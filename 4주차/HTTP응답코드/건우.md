### HTTP 상태 코드
상태코드는 3자리 숫자로 이루어져 있다.     
앞자리가 1~5 시작이 가능하며 응답코드를 5개로 분류한다.    

### 1️⃣xx = Information responses
요청을 받았으며 프로세스를 계속한다.

### 2️⃣xx = Success
요청을 성공적으로 받고 처리하였다.

### 3️⃣xx = Redirectional
요청 완료를 위해 추가 작업 조치가 필요하다.

### 4️⃣xx = Client Error
클라이언트 요청에 문제가 있음을 나타낸다.

### 5️⃣xx = Server Error
서버가 클라이언트 요청을 처리한는 동안 오류가 있음을 나타낸다.
